Steganography Application

An application for hiding messages in an image. It also allows for decryption as well if it uses the same algorithm. The images respectively show choosing an image, typing the message to encrypt, and finally the new encrypted image juxtaposed to the original. (Obviously they look identical to humans)


Wine Classification Artificial Neural Network

A neural network to classify a wine dataset into three mutually exclusive classes. First, the data is pre-processed and one hot-hot-encoded. The second image shows the creation of the neural network model and the third image is using matplotlib to visulize the results after the model has completed its epochs.

Ocufly - Drone Flight Software

Senior Seminar project. Using VCS we had to plan, create, and present our project, Ocufly - Flight and object recognition software for drones for harsh environments or dangerous situations. I was in charge of the fligh software team and unfortunatly our object recognition team was never able to implement their work because it was right when the pandemic hit.


Keylogger Application

A simple keylogger app that will diplay any interactions with the keyboard. It displays a new line for everytime a button is pressed, the output from the button/buttons, and when the key is released.
